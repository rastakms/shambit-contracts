# 🏗 Shambit DAPP

> is everything you need to get started building decentralized applications powered by smart contracts

---

## quickstart

```bash
git clone

cd 
```

```bash

yarn install

```

```bash

yarn start

```

> in a second terminal window:

```bash
cd  
yarn chain

```

> in a third terminal window:

```bash
cd  
yarn deploy

```
